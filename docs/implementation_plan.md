# Implementation Plan — премиальный фитнес-дизайн

Документ описывает UI/UX-концепцию приложения и архитектуру фронтенда:
как устроены экраны, drill-down активности и общая статистика. Это «карта»
интерфейса для разработчика, а не пошаговый туду-лист.

## Концепция UI/UX

Единый дизайн-стек для всего приложения с поддержкой нескольких активностей.

- **Стек дизайна:** Tailwind CSS (через CDN), шрифт Inter, иконки Material Symbols Outlined.
- **Вкладка «Сегодня»:** список активностей в виде карточек с прогресс-барами.
  Клик по карточке открывает детальный экран активности.
- **Детальный экран активности (drill-down):**
  - **Hero-блок:** общая сумма за всё время, показатель за сегодня, персональный
    рекорд за день и текущий streak.
  - **Weekly Progress:** недельный график со скруглёнными столбиками прогресса.
  - **Достижения:** анимированные бейджи, открывающиеся при достижении целей
    (бронза — сумма > 0; серебро — streak ≥ 3; золото — сумма ≥ 500 или 10× дневной цели).
  - **Recent Logs:** последние записи с датой/временем и удалением ошибочного лога.
  - **FAB (+):** модалка быстрого добавления лога.
- **Вкладка «Статистика»:** агрегированные графики по всем привычкам за 7/30 дней.
- **Вкладка «Активности»:** управление списком, цели, выбор цвета.

## Архитектура фронтенда

Весь фронтенд — в [public/app.js](../public/app.js), без сборки. Паттерн: `State → API → Render`.

### public/index.html
- Tailwind CSS, Inter, Material Symbols Outlined.
- Контейнеры SPA: экран входа, список целей, детальный экран, модалки.
- Единый Toast-компонент.

### public/app.js
- **State:** `state.selectedActivityId` — открытая детальная карточка.
- **Details View:** расчёт общей суммы (по всем логам активности), Personal Best
  (макс. сумма логов за один день), недельного графика (группировка по 7 дням),
  достижений и списка последних логов с удалением.
- **Today View:** список активностей карточками.
- **Activities & Modals:** Tailwind-модалки с анимациями.

## Верификация

Ручное тестирование локально (`npm run dev`):

1. Зарегистрировать пользователя — главный экран показывает активности в новом дизайне.
2. Открыть активность «Подтягивания» — открывается детальный дашборд.
3. Добавить лог через FAB — графики, Today, Personal Best и Total мгновенно пересчитываются.
4. Проверить разблокировку достижений при достижении лимитов.

Автоматические E2E-сценарии — см. [tests/README.md](../tests/README.md).
