# План интеграции премиального фитнес-дизайна в trecker

Мы получили великолепный референс дизайна страницы активности (сводка, недельный график прогресса, ачивки, история логов, плавающая кнопка FAB, современная палитра и Material Symbols). Мы интегрируем этот стиль во всё приложение, сохранив поддержку нескольких активностей.

## Новая концепция UI/UX:
1. **Основной стек дизайна:** Переход на Tailwind CSS (используем Tailwind CDN с плагинами, как в референсе) для молниеносной разработки и 100% совпадения с премиальной темой.
2. **Вкладка «Сегодня» (Сегодняшний трекер):**
   - Отображает список всех ваших активностей в виде красивых современных карточек с прогресс-барами.
   - Клик по карточке активности открывает **детальный экран этой активности** (в точности как в вашем референсе!).
3. **Детальный экран активности (Drill-down):**
   - **Hero-блок:** Общая сумма выполненного за все время («Total Reps/Pages/Minutes»), показатель за сегодня, персональный рекорд за день и текущий Streak.
   - **Интерактивный график Weekly Progress:** Недельный график с закругленными столбиками прогресса.
   - **Достижения (Achievements):** Раздел с анимированными плавающими бейджами (например, «100 Club», «Elite Level», «Consistent King»), которые открываются при достижении целей.
   - **История последних записей (Recent Logs):** Список последних логов с указанием даты, времени, сетов и возможностью удалить ошибочный лог.
   - **FAB (Плавающая кнопка +):** Вызывает красивое модальное окно для быстрого добавления лога.
4. **Вкладка «Статистика» (Общий прогресс):**
   - Агрегированные графики по всем привычкам за 7/30 дней.
5. **Вкладка «Активности» (Управление привычками):**
   - Управление списком активностей, добавление, редактирование целей, выбор цвета с поддержкой новой палитры.

## Предлагаемые изменения

### 1. public/index.html [MODIFY]
- Перевод на Tailwind CSS, шрифты Inter и иконки Material Symbols Outlined.
- Добавление контейнеров для SPA: экрана входа, экрана списка целей, детального экрана выбранной цели и модалок.
- Реализация единого Toast-уведомления в новом стиле.

### 2. public/app.js [MODIFY]
- **State:** Добавим `state.selectedActivityId` для отслеживания открытой детальной карточки.
- **Render Details View:**
  - Расчет общей суммы (суммирование по всем логам этой активности).
  - Расчет Personal Best (поиск максимальной суммы логов за один день).
  - Генерация недельного графика прогресса (группировка логов по последним 7 дням).
  - Отрисовка достижений:
    - Бронзовый бейдж: если общая сумма > 0.
    - Серебряный бейдж: если streak >= 3.
    - Золотой бейдж: если общая сумма >= 500 (или 10x от дневной цели).
  - Отрисовка списка логов (recent logs) с возможностью удаления по кнопке.
- **Render Today View:** Список активностей в виде премиальных карточек.
- **Render Activities & Modals:** Современные Tailwind-модалки с анимациями открытия/закрытия.

## План верификации

### Ручное тестирование локально
1. Запустим локальный сервер `npm run dev` (он уже запущен).
2. Зарегистрируем пользователя и проверим, что главный экран отображает список активностей в новом дизайне.
3. Кликнем на активность "Подтягивания" -> откроется шикарный детальный дашборд.
4. Добавим лог через FAB-кнопку -> проверим, что графики, Today, Personal Best и Total Reps мгновенно пересчитались.
5. Проверим разблокировку достижений при достижении нужных лимитов.
